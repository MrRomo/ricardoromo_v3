Felicidades a todos los que llegaron hasta esta parte del tutorial, ahora vamos a ver en acci\u00f3n el proyecto final y como puedes ejecutarlo.\nComo saben dentro de los requisitos de este proyecto, era necesario tener un sistema linux ya que el programa utiliza librer\u00edas propias del sistema y en windows habr\u00eda que hacer algunos cambios, en la pr\u00f3xima parte del tutorial explicare como hacer estos cambios para que tambi\u00e9n funcione en windows.\nPrimero que todo, hay que compilar el programa nos dirigimos a la carpeta del proyecto, abrimos una terminal y tecleamos los siguientes comandos.\n<\/pre>\t\t\t\t\t<\/div>\n\t\t\t\t\t\t<\/div>\n\t\t\t\t<\/div>\n\t\t\t\t
